This repository contains the documents for our structured literature review (SLR), blinded for our FAccT 2026 submission.
It contains 
- An overview of all screened data (20251125_SLR_Overview.csv)
- Our exact search queries (Literatur_Prisma_FAccT.txt)
- Our AI reporting content analysis results (Model_Card_Evaluation_VisualEvaluation.csv)
- Our final and interim taxonomies including categorized papers for each round (Export_Appendix_Interim_Taxonomy.csv; Export_Appendix_Final_Taxonomy.csv)
